# Vendora Marketplace 🚀

Vendora is a high-performance, vibrant e-commerce marketplace built for the modern era. Inspired by global marketplaces like Temu and Amazon, it features a mobile-first UI, real-time data synchronization, and automated seller-platform earnings splitting.

## Key Features
- **Flash Deals**: Real-time discounted items with countdown timers.
- **Dynamic Marketplace**: Category-based browsing and powerful search.
- **Secure Payments**: Paystack integration with automated 80/20 split logic.
- **Seller Ecosystem**: Dedicated dashboard for inventory management and earnings tracking.
- **Admin Control**: Robust system for seller verification and platform analytics.
- **Role-Based Access**: Specialized views for Buyers, Sellers, and Admins via Supabase RLS.

## Prerequisites
- **Node.js**: v18.0 or higher
- **Supabase Account**: For database, auth, and edge functions
- **Paystack Account**: For payment processing
